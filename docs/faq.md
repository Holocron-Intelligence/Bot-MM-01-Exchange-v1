# ❓ FAQ

### 1. "The bot says IDLE, why isn't it trading?"
*   Check if **01 Exchange** is online in your browser.
*   Ensure you have selected at least one coin in the launcher.
*   Verify that you have SOL in your wallet for transaction fees.

### 2. "Does the bot work on Mac or Linux?"
*   Currently, the Launcher is optimized for **Windows**. However, since it is Python-based, it can run on Mac/Linux if you have the dependencies installed.

### 3. "Can I run multiple versions at once?"
*   **Yes!** We have specifically updated the system so you can run the Stable, Optimized BE, and Hyper Volume versions simultaneously on different ports (8000, 8001, 8002).

### 4. "What is the minimum capital?"
*   Technically, the bot can run with as little as $10. However, for efficient volume farming, we recommend **$50 - $100** per coin selected.

### 5. "Why did the bot close my position at a small loss?"
*   This is likely the **Stale Position Manager** or a **Stop-Loss**. It's better to take a tiny 0.1% loss and resume farming than to stay stuck in a bad trade for hours.
